# Challenge Mercado Libre
## Cómo iniciar el proyecto?
```bash
git clone https://github.com/FacuEM/challenge-meli.git
```
### FRONT 
```bash
cd challenge-meli/front  
npm install
npm run build
```
### BACK 
```bash
cd challenge-meli/back  
npm install
npm start
```
## Tecnologías utilizadas:
+Database: Sqlite3
+ORM: Sequelize
+Back-end:
..+Node.js
..+Express.js
+Front-end;
..+React.js
..+React-redux
+Styles: Material UI
